*** Belajar Github ***
# # # SIGN UP # # #
- sign up -> email (pastikan gunakan email yang belum pernah terdaftar)
- create folder/repository baru kalian
- gunakan visible public, jika ingin repo kita dapat dilihat oleh orang lain
- gunakan visible private, jika ingin repo tidak dapat terlihat oleh orang lain
- klik create repository

*** membuat file ***
- isikan nama file (bisa menggunakan format extension apapun)
- menuliskan isi dari file tersebut
- berikan nama commit yang sesuai
- commit changes

*** lakukan branching / percabangan commit dari main/master branch ***
- terdapat 2 cara
